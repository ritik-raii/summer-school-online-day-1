About Me

Hello! I’m Ritik Rai, a tech enthusiast from Varanasi, Uttar Pradesh, currently pursuing a B.Tech in Computer Science and Engineering from Rajkiya Engineering College, Mainpuri (affiliated to Dr. A.P.J. Abdul Kalam Technical University).

I completed my schooling at Sant Atulanand Convent School, Varanasi with 84.2% in Class 12. Ever since then, I’ve been passionate about solving real-world problems using technology and have explored fields like IoT, Machine Learning, and Web Development.

Technical Skills & Interests

  • Programming: Python, C, C++, HTML, CSS
  • Machine Learning: Scikit-learn, Pandas, NumPy, Matplotlib
  • IoT Systems: ESP8266, Blynk, ThingSpeak, Sensors
  • Tools: GitHub, Google Colab, VS Code
  • Soft Skills: Communication, Leadership, Teamwork, Problem-Solving

Projects I’m Proud Of

  1. Sahayak Khata: A personal finance manager built with HTML, CSS, and JavaScript that includes a login system, dashboard, and local storage support.
  2. WISER Weather Box: A weather monitoring system using ESP8266 and sensors to track temperature, humidity, and pollution with real-time cloud dashboards.
  3. Shear-Capacity-ML: A machine learning model to predict steel beam shear capacity, trained using real data and fine-tuned preprocessing.
